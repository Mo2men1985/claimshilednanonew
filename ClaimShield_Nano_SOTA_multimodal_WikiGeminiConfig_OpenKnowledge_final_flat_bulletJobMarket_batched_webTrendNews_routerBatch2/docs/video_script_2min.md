# 2-minute demo script (read this verbatim)

(0:00–0:30) Verify in context
Right-click a paragraph → “Verify with ClaimShield.” Show ProofCard with citations.
Line: “Local-only mode — privacy-first. If evidence is weak, we abstain.”

(0:30–1:00) Hybrid
Popup → toggle Hybrid ON → paste key → Verify again.
Line: “Hybrid strengthens retrieval for tough claims—user-controlled.”

(1:00–1:20) Built-in AI TL;DR
Popup → “Open Built-in TL;DR” → paste text → Run.
Line: “Uses Chrome’s on-device Summarizer; availability is shown here.”

(1:20–1:35) Diagnostics
Popup → “Open Diagnostics” → Run.
Line: “We show Wikipedia and Gemini connectivity—no silent errors.”

(1:35–2:00) Export
Popup → “Export ProofCard (MD)” → paste or load → Generate.
Line: “Proof-carrying, privacy-first, hybrid-smart.”
