# ClaimShield Nano – AI Genesis Build

## How to run
1. Open Chrome → `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.
4. Pin the extension.
5. Select any text on a web page → right-click → **Verify with ClaimShield**.
